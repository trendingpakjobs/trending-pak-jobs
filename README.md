# # Trending Pak Jobs

**Trending Pak Jobs** is a professional website showcasing the latest job opportunities across Pakistan.  
It provides daily updated job listings in different categories: Government, Private, Internships, and Online / Work From Home jobs.  

---

## Features

- **Category-wise listings**: Government, Private, Internships, Online Jobs  
- **Search and Filter by City**: Easily find jobs in your preferred city  
- **Latest Jobs on Home Page**: Displays newest jobs at a glance  
- **Professional Job Cards**: Include job title, company, salary, education, requirements, rules, and apply button  
- **Responsive Design**: Mobile and desktop friendly  
- **Green & White Theme**: Inspired by Pakistan flag for a clean and professional look  
- **About Section**: Brief description of each category for user clarity  

---

## Pages Included

- `index.html` – Home page showing latest jobs  
- `government-jobs.html` – Government job listings

- `internships.html` – Internship opportunities  
- `online-jobs.html` – Work from home / online jobs
